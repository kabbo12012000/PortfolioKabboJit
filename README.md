# Kabbo Jit Deb - Cinematic Portfolio

A single-page, cinematic portfolio website for Kabbo Jit Deb (AI/ML researcher, DTU '26). Built with pure HTML, CSS, and vanilla JavaScript. Designed for a dark, neon-accented, glassmorphic aesthetic with subtle motion.

## Live Demo
- https://kabbo12012000.github.io/PortfolioKabboJit/

## Highlights
- 3D rotating cube hero with image-mapped faces
- Parallax background text with scroll animation
- Glassmorphism cards, neon hover accents, and custom cursor
- IntersectionObserver reveal animations with staggered entry
- Fully responsive layout for mobile and tablet

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Font: General Sans (Fontshare)

## Run Locally
1. Download or clone the repository.
2. Open `index.html` in your browser.

No build step required.

## Customize
- Update text content in `index.html`.
- Replace images in `images/` and adjust URLs in `styles.css`.
- Tweak animations and layout in `styles.css`.

## Deploy (GitHub Pages)
1. Push to GitHub.
2. Go to **Settings -> Pages**.
3. Source: **Deploy from a branch**.
4. Branch: `main` and **/(root)**.
5. Save and wait for the URL to go live.
